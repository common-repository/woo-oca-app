Obtener costos de envio de OCA sin tener cuenta con ellos.

== Descripcion ==
Con este plugin vas a poder obtener costos de envio de OCA, sin tener que crear una cuenta con ellos.


== Instalacion ==

1. Instalar el plugin APP de OCA para WooCommerce.
2. Activar el plugin.
3. Ir a WooCommerce -> Ajustes.
4. Ir a ENVIOS.
5. Si ya tenes una zona de envio ingresa en la misma, sino, crea una zona de envio.
6. Dentro de la zona de envio, agrega el metodo OCA.
7. Ingresa en la configuracion del plugin y carga los datos solicitados.
8. Recorda tener tus productos con PESO Y DIMENSIONES, asi OCA puede calcular el costo de envio.
